# HdLabs.Common

HdLabs 데스크톱 앱에서 공유하는 **WPF MVVM·UI 리소스** 라이브러리입니다.

## 프로젝트 분석

- **MVVM:** `ObservableObject`, `RelayCommand` / `RelayCommand<T>`, `AsyncRelayCommand`, `ViewModelBase`
- **UI:** `UI/Theme.xaml` (공통 브러시·카드·버튼 스타일) — `App.xaml`에서 MergedDictionary로 참조
- **역할:** 앱마다 **동일 룩앤필**·**명령 패턴** 유지

## 기능 점수 (10점 만점, “라이브러리” 기준)

| 항목 | 점수 | 비고 |
|------|------|------|
| **재사용성** | **8** | 복붙·확장 용이 |
| **기능 풍부함** | **4** | 심플—고급 DI/로깅 없음 |
| **종합(소규모 팀)** | **7** | 과하지 않은 공통층 |

## 확장성

- **IUserDialog**, **ISettingsStore** 인터페이스·서비스 추출로 **테스트** 개선
- **CommunityToolkit.Mvvm**로 이전 시 보일러플레이트 감소

## 장점

- **빌드/참조 단순** (단일 .csproj)
- 신규 앱 **톤 통일**에 유리

## 단점

- **버전 올릴 때** 모든 앱 **동시** 검증 권장(깨지기 쉬움)
- **테마**가 커질수록 **느슨한 결합** 필요

## 제품으로서의 매력(내부)

직접 팔리는 “제품”이라기보다 **브랜드 일체감**·**개발 비용 절감**용 **엔지니어링 자산**입니다. **HD Labs**라는 **한 세트**로 말할 수 있게 해 주는 **접착제**에 가깝습니다.

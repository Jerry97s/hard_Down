# HdLabs.Cam (EzLabs Cam)

**웹캠 프리뷰/촬영**을 염두에 둔 WPF 셸입니다. 현재는 **Ready** 상태 표시 수준의 **스캐폴딩**이며, 카메라 열거·렌더링·녹화는 **미구현**입니다.

## 프로젝트 분석

- **후보 기술:** MediaCapture(WinRT interop), OpenCV, DirectShow 래퍼 등—**선택 전**
- **과제:** 권한(UVC)·해상도·프리뷰 지연·배터리

## 기능 점수 (10점 만점)

| 항목 | 점수 |
|------|------|
| **현재** | **2~3** |
| **(캠 프리뷰+스냅 구현 시)** | **6~8** |

## 확장성

- **가상 배경**·**필터**는 별도 파이프라인
- **Memo**와 합치면 **화상 메모·스크린캠** 시나리오

## 장점

- **시리즈 통합** 용이
- 원하면 **회의·교육** 니치로 스핀 가능

## 단점

- **기능 공백** — OS·브라우저 기본 카메라앱과 비교 시 **차별** 필요
- **프라이버시** 고지·권한 UX 필수

## 제품으로서의 매력

**“PC에서 가볍게 캠 확인”** 보조. **스탠드얼론**보다 **HD Labs 번들(캡처+메모)** 의 **하나의 렌즈**로 묶는 편이 설득력이 큼.

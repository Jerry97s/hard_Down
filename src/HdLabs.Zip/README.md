# HdLabs.Zip (EzLabs Zip)

**ZIP/압축 유틸**을 위한 WPF 셸입니다. 현재 ViewModel은 **Ready 상태·표시** 수준의 **스캐폴딩**에 가깝고, 압축/해제 파이프라인은 **미구현**입니다.

## 프로젝트 분석

- **기술:** WPF, `RelayCommand`로 UI 바인딩, 실제 I/O·SharpZipLib/ZipFile API 연동은 추가 예정
- **역할:** 제품 **프레임**·브랜딩 자리

## 기능 점수 (10점 만점)

| 항목 | 점수 |
|------|------|
| **현재** | **2~3** |
| **확장 후(동일 셸)** | **6~7** (기능만 채웠을 때 체감) |

## 확장성

- `System.IO.Compression` / 서드파티로 **한 창**에 “압축/풀기/목록” 탭을 나누기 좋음
- **드래그앤드롭**·**비밀번호 ZIP**·**다중 볼륨** 순으로 확장 가능

## 장점

- **아이콘·Theme**이 시리즈와 통일
- **기능이 비어** 오히려 **요구정의**에 백지에 가깝게 맞출 수 있음

## 단점

- **사용자 가치 0**에 가깝(현상태) — PoC/내부용 명시 필요

## 제품으로서의 매력

**“한국 PC 유저 친화 Zip 도구(광고 없음/가벼움)”** **세일즈 스토리**는 가능. 다만 **7-Zip/윈도 기본**과 비교 **차별화**(배치·클라우드·프리셋)가 없으면 **대체재**로 보이기 쉬움.

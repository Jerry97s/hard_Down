# HdLabs.Finder (EzLabs Finder)

**파일 이름 기반 검색**을 제공하는 WPF 앱입니다. 쿼리 입력 후 비동기로 스캔하며, 결과에서 파일 열기·폴더 열기를 지원합니다.

## 프로젝트 분석

- **기술:** `AsyncRelayCommand`, `CancellationToken`으로 검색 취소, 사용자 프로필 하위 위주 스캔(구현에 따라 제한)
- **UI:** 카드 스타일 리스트, 상태 텍스트
- **한계(코드/주석 기준):** 전역 인덱싱·즉시 검색은 “다음 단계”로 남겨 둔 상태

## 기능 점수 (10점 만점)

| 항목 | 점수 | 비고 |
|------|------|------|
| 검색 (현재) | **6** | 동작하나 Everything 대비 느낌·범위 제한 |
| UX | **6** | 단순 명료 |
| **종합** | **6** | **기능 확장 전** 제품 점수 |

## 확장성

- **인덱서/백그라운드 갱신**만 넣으면 체감 품질이 크게 오름
- **필터(확장자·날짜)** · **즐겨찾기 경로** 추가에 적합

## 장점

- **비동기 + 취소**로 UI 멈춤 완화
- 공용 **테마/카드**와 일관

## 단점

- **전역·실시간** 검색 수준은 아님(제품 기대치 관리 필요)
- **내용(본문) 검색** 없음(파일명 중심)

## 제품으로서의 매력

**“가벼운 파일 찾기”** 보조 도구. 개발자·PC 관리담당이 **로컬만** 쓰는 2nd Finder 포지션이 자연스럽습니다. 인덱싱만 갖추면 **“설치부담↓ Everything 대안 시도”** 내러티브 가능.

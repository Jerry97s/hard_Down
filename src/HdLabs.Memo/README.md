# HdLabs.Memo

스티키 노트 느낌의 **Windows 데스크톱 메모** 앱입니다. 투명 창, 민트 톤 UI, 툴바·목록·편집기를 한 카드에 묶었습니다.

## 프로젝트 분석

- **기술 스택:** .NET 9, WPF, MVVM(`HdLabs.Common`), JSON 파일 영속
- **데이터:** `%LocalAppData%\HdLabs\Memo\memos.json` (메모 목록 + 항상 위·카드 틴트 등 설정)
- **구조:** `MainViewModel`에서 목록/편집 뷰 전환, `MemoDataService`로 저장/로드, `MemoItem`이 `ObservableObject`로 목록 UI와 동기화

## 기능 점수 (10점 만점)

| 항목 | 점수 | 비고 |
|------|------|------|
| 핵심 가치 (빠른 메모) | **8** | CRUD, 자동 저장, 단축키 |
| UI 완성도 | **7** | 그라데이션·구분선·슬라이더 등 다듬음 |
| 안정성 | **6** | 단일 PC·단일 프로세스 기준, 동시 편집 없음 |
| **종합(체감)** | **7.5** | 실사용 메모앱으로 쓰기엔 양호 |

## 확장성

- **좋은 점:** JSON 스키마 확장(태그, 동기화 플래그), RichText, 클라우드 API 연동에 구조를 덧댈 수 있음
- **주의:** 본문이 평문+마커 수준—리치텍스트·첨부는 별도 설계 필요

## 장점

- 설치 느낌 없이 가벼운 **로컬 우선** 메모
- **눈에 편한** 색·투명도·항상 위 등 PC 작업 보조에 맞춤
- **공용 라이브러리**와 분리되어 테스트·빌드가 단순

## 단점

- **클라우드/모바일 동기화 없음** (의도에 따라 중립 또는 단점)
- **일정/알림**은 UI·메시지 수준이라 실기능은 추후
- **멀티 유저/충돌** 미고려

## 제품으로서의 매력

**“자리 차지 않는 메모지”** 포지션이 분명합니다. OS 메모앱이 무겁게 느껴질 때, HD 계열 톤으로 **업무/개인 감정 분리**를 시각적으로 주는 데 잘 맞습니다. B2C로는 “미니멀+로컬” 니치, 사내则 “개인 TODO·임시 기록” 도구로 설명하기 좋습니다.
